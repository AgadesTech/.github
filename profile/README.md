# Agades Technologies

Agades Technologies is a research company building AI systems for cryptographic security.

We study attack spaces as search problems: lattices, candidate strategies, program structure, estimators, proofs, and reproducible evaluation loops. Our public work focuses on post-quantum cryptanalysis research and evaluator-driven evolutionary program synthesis.

## Research Direction

- **Post-quantum cryptanalysis**: exploring LWE and MLWE attack-strategy spaces on toy, downscaled, and public benchmark settings.
- **Evolutionary program synthesis**: searching for structured candidate strategies rather than known signatures.
- **Formal evaluation**: treating estimators, proofs, traces, and reproducibility as first-class research objects.
- **Responsible security research**: publishing tools and artifacts that support analysis without claiming unsupported breaks or targeting unauthorized systems.

## Principles

- Search before assertion.
- Evaluation before claim.
- Reproducibility before scale.
- Responsible disclosure before publication.
